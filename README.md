# Project2JavaScript
A task on the U.S. National Park Service Data Analysis Practice. 🇺🇸
Completed in VS Code. 

This mini-project covers concepts such as creating Objects from Arrays, loops with a Standard for Loop, using map(), Use filter(). As well as dot notation and bracket notation. 

It was a fun activity and I am thinking of completing an additional Naional Park Analysis for the parks in South Africa. 🇿🇦
